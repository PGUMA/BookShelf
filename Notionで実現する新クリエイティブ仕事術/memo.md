# インプットについて

【ポイント１】

情報　→　思考　→　インテリジェンス

情報に自身の思考を通してインテリジェンスに変える

【ポイント２】

具体　→　抽象

幅広い分野から情報を集めて、共通項を見出して抽象化して応用を効かせる

【ポイント３】

後での検索性を意識して溜める
